红米Turbo 4（不可刷机）

设备说明

红米Turbo 4是我的日常主力机，这款机型目前官方未开放BootLoader解锁权限，且第三方也没有适配的解锁工具，因此暂时无法进行刷机、ROOT等操作，只能使用官方原版系统。

使用体验

虽然不能刷机，但原版MIUI系统流畅度不错，电池续航也能满足日常上学使用，平时主要用它来学习、社交和玩轻度游戏。

红米Note 8 Pro（可刷机）

设备概况

红米Note 8 Pro是我的刷机专用备用机，搭载联发科G90T处理器，官方已开放BootLoader解锁权限，第三方Recovery和ROM资源也比较丰富，非常适合用来练手刷机。

刷机优势

1. 解锁流程简单，小米官网可直接申请解锁；
2. 第三方ROM选择多，如LineageOS、MIUI开发版、Pixel Experience等；
3. 硬件配置足够，刷入各类系统后运行流畅。

红米Note 8 Pro解锁申请

申请步骤

1. 打开小米解锁官网（https://www.miui.com/unlock/），登录绑定手机的小米账号；
2. 点击“解锁申请”，填写手机型号（红米Note 8 Pro）、IMEI码、手机号等信息；
3. 提交申请后，等待官方审核，通常1-3天即可通过（小米账号需登录手机7天以上）；
4. 审核通过后，即可下载小米解锁工具进行下一步操作。

注意事项

申请时需确保手机与小米账号绑定，且账号在手机上连续登录满7天，否则审核会失败。

返回上一级

红米Note 8 Pro工具解锁

操作步骤

1. 下载并安装小米解锁工具，打开后登录审核通过的小米账号；
2. 手机进入开发者模式，开启“OEM解锁”和“USB调试”，然后关机；
3. 按住电源键+音量下键进入fastboot模式，用数据线连接电脑；
4. 点击解锁工具中的“解锁”按钮，等待进度条完成（约1分钟），解锁成功后手机会自动重启。

重要提醒

解锁会清除手机所有数据，操作前务必备份照片、文件等重要内容；解锁后手机将失去官方保修。

红米Note 8 Pro开发版固件下载

下载步骤

1. 打开小米ROM官网（https://rom.miui.com/），选择“红米Note 8 Pro”机型；
2. 切换到“开发版”标签，查看最新的开发版固件版本，点击“下载”；
3. 固件为zip压缩包，下载完成后无需解压，记住保存路径（后续刷机工具需要用到）。

版本选择

建议选择最新的稳定开发版，避免选择测试版，防止出现系统bug影响使用。

红米Note 8 Pro MiFlash刷机

操作步骤

1. 下载并安装MiFlash刷机工具，安装时需勾选安装驱动（确保手机能被电脑识别）；
2. 手机进入fastboot模式，连接电脑，打开MiFlash工具，点击“加载设备”识别手机；
3. 点击“选择”，找到下载好的MIUI开发版固件文件夹；
4. 选择“清除所有数据”模式，点击“刷机”，等待进度完成（约5分钟），刷机成功后手机会自动重启。

常见问题

若刷机时提示“设备未识别”，需重新安装小米驱动，或更换数据线/电脑USB接口；刷机后卡在开机界面，可重新进入fastboot模式再次刷机。

红米Note 8 Pro线刷救砖

适用场景

手机变砖后无法开机、无法进入Recovery，仅能进入fastboot模式时使用。

操作步骤

1. 下载红米Note 8 Pro官方稳定版固件和MiFlash工具；
2. 手机进入fastboot模式，连接电脑，打开MiFlash工具加载官方固件；
3. 选择“清除所有数据”，点击“刷机”；
4. 刷机完成后手机自动重启，即可恢复官方系统，完成救砖。

红米Note 8 Pro卡刷救砖

适用场景

手机能进入Recovery模式，但无法正常开机时使用。

操作步骤

1. 下载红米Note 8 Pro官方卡刷包，放入手机SD卡根目录；
2. 手机关机，按住电源键+音量上键进入官方Recovery；
3. 选择“安装更新包”，找到SD卡中的卡刷包，确认刷入；
4. 刷入完成后，选择“重启手机”，即可恢复系统。

ROOT准备工作

步骤1：提取boot.img

1. 下载与手机当前系统版本一致的MIUI固件包；
2. 用解压软件打开固件包，提取里面的boot.img文件；
3. 将boot.img复制到手机存储空间中。

步骤2：下载Magisk

1. 打开Magisk官网下载最新版Magisk APK安装包；
2. 在手机上安装Magisk APK，打开后点击“选择并修补一个文件”；
3. 选择刚才复制的boot.img，点击“修补”，等待生成修补后的magisk_patched.img文件。

返回上一级

刷入Magisk获取ROOT

操作步骤

1. 将修补后的magisk_patched.img文件复制到电脑；
2. 手机进入fastboot模式，连接电脑；
3. 在电脑命令提示符中输入：fastboot flash boot magisk_patched.img；
4. 输入：fastboot reboot 重启手机，打开Magisk APP，即可完成ROOT。

注意事项

刷入前需确保手机已解锁BL且安装了第三方Recovery，否则会刷入失败。

红米Note 8 Pro ROOT验证

验证方法1：Magisk APP

打开手机上的Magisk APP，如果显示“安装状态：已安装”，且版本号正常，说明ROOT成功。

验证方法2：ROOT检测工具

下载安装“Root Checker”APP，打开后点击“验证ROOT权限”，如果提示“设备已获得ROOT权限”，则ROOT成功。

下载适配的TWRP Recovery

下载步骤

1. 打开TWRP官网或第三方安卓刷机论坛；
2. 搜索“红米Note 8 Pro TWRP”，选择最新版本的TWRP镜像文件（格式为.img）；
3. 将下载好的twrp.img文件复制到电脑，记住保存路径。

版本选择

选择标注“支持红米Note 8 Pro”且发布时间较新的版本，避免老旧版本出现兼容性问题。

返回上一级

刷入TWRP Recovery

操作步骤

1. 手机进入fastboot模式，连接电脑；
2. 在电脑命令提示符中输入：fastboot flash recovery twrp.img（需替换为你的TWRP文件名）；
3. 输入：fastboot reboot recovery，手机会重启进入TWRP界面；
4. 首次进入TWRP会提示是否允许修改系统，选择“保留只读”即可。

验证方法

重启进入Recovery后，若显示TWRP的操作界面，说明刷入成功。

卸载预装APP

操作步骤

1. 手机获取ROOT权限后，安装“系统程序卸载器”APP；
2. 打开APP，授予ROOT权限，会列出所有系统预装APP；
3. 选择无用的APP（如小米视频、小米音乐、游戏中心等），点击“卸载”；
4. 卸载完成后重启手机，系统会更流畅。

注意事项

不要卸载核心系统APP（如设置、拨号、短信等），否则会导致手机变砖。

返回上一级

禁用冗余系统服务

操作步骤

1. 安装“Servicely”APP，授予ROOT权限；
2. 打开APP，会显示所有正在运行的系统服务；
3. 禁用无用服务（如谷歌位置服务、自动同步服务、广告推送服务等）；
4. 禁用后重启手机，可减少后台耗电和内存占用。

注意事项

禁用服务前需确认服务功能，避免禁用关键服务导致系统异常。

TWRP卡刷第三方ROM

操作步骤

1. 下载第三方ROM包，复制到手机SD卡根目录；
2. 手机关机，按住电源键+音量上键进入TWRP Recovery；
3. 选择“清除”，勾选data、cache、dalvik分区，点击“滑动清除”；
4. 选择“安装”，找到ROM包，滑动确认刷入；
5. 刷入完成后，选择“重启系统”，首次开机需等待5-10分钟。

注意事项

刷入第三方ROM前需备份重要数据，部分ROM需要刷入谷歌服务包，可根据需求选择。
